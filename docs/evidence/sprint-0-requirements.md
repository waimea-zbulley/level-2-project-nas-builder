# Project Requirements

## Identified Problem or Need

Picking out the components and configuration for a home NAS is difficult and tedious. Especially enusing all parts are compatible for the configuration that you want to run.


## End-User Requirements

For my end user I need to make a system that allows you to pick all the hardware for a NAS in a nice simple interface. For the configuration of the NAS you will need to be able to pick raid configuration (and weather you need redundancy), the security for the NAS, and recomendations based on what the NAS is primarly used for (e.g. Media server, raw storage...).

Features needed:
 - Being able to select hardware
 - Selecting the desired raid setup and amount of desired usable storage.
 - Being able to save specfic configured setups
 - Being able to select the desired software on the system
 - Only being able to choose parts which are compatable with each other
 - Make recomendation based off of the usecase of the NAS


## Proposed Solution

I will make a website which takes you through the processes of selecting the right hardware (and software) for your needs in a NAS, and that guides you through the process.
